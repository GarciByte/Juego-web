# Juego de Memoria

Este proyecto es una aplicación web desarrollada con **ASP.NET** en el backend y **Angular** en el frontend. La aplicación permite a los usuarios registrarse, iniciar sesión, gestionar sus perfiles, buscar y añadir amigos y jugar en línea al **Juego de Memoria (Parejas)**.

Enlace a la aplicación desplegada: [Juego de Memoria](https://memory-game-david.vercel.app).
